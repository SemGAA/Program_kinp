# Cinema Notes v1.5.9-beta.1

Первая оформленная beta-сборка с новым GitHub-оформлением, релизным потоком и исправлениями мобильного сценария просмотра.

## Что изменилось

- Обновлена комната просмотра: плеер больше не съедает чат, экран стал стабильнее на Android.
- Нижняя навигация учитывает safe-area и не должна перекрываться системными кнопками.
- Добавлен каталог для быстрого входа в поиск аниме, фильмов, сериалов, открытых видео и Jellyfin.
- Поиск больше не открывает пустую комнату, если у результата нет встроенного источника.
- Добавлена профессиональная GitHub Page и обновлён README.
- Добавлены changelog, copyright notice и release workflow.
