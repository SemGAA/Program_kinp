# Cinema Notes

Cinema Notes — мобильное Android-приложение для заметок о фильмах, комнат совместного просмотра, друзей, профилей и чата рядом с видео.

## Что уже есть

- Регистрация и вход с подтверждением email-кодом через API приложения.
- Профиль с аватаркой, никнеймом, описанием и публичной страницей пользователя.
- Друзья, заявки, приглашения в комнаты и удаление/скрытие комнат.
- Живой поиск по каталогам: TMDB для фильмов/сериалов, Shikimori для аниме, Internet Archive для открытых видео и Jellyfin для личной медиатеки.
- Комнаты совместного просмотра с синхронизацией прямых потоков, чатом, заметками и красивым текстом приглашения.
- Заметки теперь открывают редактор заметки, а не экран поиска.
- Адаптивная нижняя навигация с safe-area для Android.

## Просмотр видео

Приложение не подключает пиратские балансеры. Встроенный плеер работает с легальными источниками: открытыми видео Internet Archive, прямыми видеофайлами и вашей личной Jellyfin-медиатекой. Если найден только каталог без потока, приложение открывает карточку/заметку и не ведёт в пустой плеер.
