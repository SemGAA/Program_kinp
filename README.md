# Cinema Notes

Мобильное Android-приложение для совместного просмотра, заметок, друзей и чата рядом с видео.

[Скачать APK](https://raw.githubusercontent.com/SemGAA/Program_kinp/apk-downloads/downloads/cinema-notes-android.apk) · [Релизы](https://github.com/SemGAA/Program_kinp/releases) · [Страница проекта](https://semgaa.github.io/Program_kinp/)

## Зачем это приложение

Cinema Notes закрывает простой сценарий: выбрать фильм, сериал или аниме, создать комнату, пригласить друга и обсуждать просмотр в одном месте. Приложение не заставляет искать комнату вручную, копировать случайные ссылки или держать чат отдельно от видео.

## Что внутри

- Живой поиск: варианты появляются во время ввода.
- Каталог: фильмы, сериалы, аниме, открытые видео и личная Jellyfin-медиатека.
- Комнаты просмотра: код комнаты, шаринг, участники, приглашения друзьям.
- Плеер: встроенный просмотр для прямых потоков, Internet Archive и Jellyfin.
- Чат рядом с видео: сообщения не уводят пользователя с экрана просмотра.
- Заметки: отдельный редактор впечатлений и рекомендаций.
- Профили: аватарка, никнейм, описание, друзья и публичная страница.
- Android UI: safe-area навигация, русская локализация и адаптация под клавиатуру.

## Источники видео

Cinema Notes не подключает пиратские балансеры. Каталог может находить карточки через TMDB и Shikimori, но встроенный плеер запускает только подключённые легальные или личные источники:

- Internet Archive для открытых видео.
- Прямые видеофайлы `mp4`, `m3u8`, `webm`.
- Jellyfin для личной библиотеки пользователя.

Если у найденной карточки нет потока, приложение открывает карточку и заметки, а не пустой сломанный плеер.

Подробная политика источников: [docs/source-policy.md](./docs/source-policy.md).

## Быстрый старт для разработки

```bash
cd cinema-app
npm install
npm run lint
npx tsc --noEmit
```

Сборка локального Android APK:

```bash
cd cinema-app/android
./gradlew assembleRelease --console=plain
```

## Структура

- `cinema-app` — Expo/React Native приложение.
- `cloudflare-worker` — API для аккаунтов, комнат, друзей, заметок и приглашений.
- `docs` — GitHub Page приложения.
- `release-notes` — тексты для GitHub Releases.
