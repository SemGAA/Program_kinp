# Changelog

Все заметные изменения Cinema Notes фиксируются здесь.

Формат версий:

- `vX.Y.Z` — стабильный релиз.
- `vX.Y.Z-beta.N` — тестовая сборка для проверки на устройствах.

## v1.5.9-beta.1 — 2026-04-22

### Главное

- Переработана GitHub Page как продуктовая страница приложения.
- Добавлен релизный поток: beta/stable теги, release notes и workflow для GitHub Releases.
- Обновлена страница README: понятное описание, APK, каналы версий, источники видео и авторские права.

### Android-приложение

- Исправлена компоновка комнаты просмотра: плеер больше не ломает высоту чата.
- Чат в комнате просмотра стал ближе к сценарию “смотрим и общаемся”.
- Нижняя навигация учитывает safe-area Android и не должна перекрываться системными кнопками.
- Добавлен экран каталога для аниме, фильмов, сериалов, открытых видео и Jellyfin.
- Поиск больше не открывает пустую комнату, если у карточки нет доступного встроенного источника.

### Проверки

- `npm run lint`
- `npx tsc --noEmit`
- `gradlew assembleRelease --console=plain`
